# Vocabulary Memory App

An AI-powered English vocabulary learning app designed for Arabic learners.

## Features

- Save English words
- English explanations
- Arabic translation
- Pronunciation audio
- Word family generation
- Memory tricks
- Example sentences
- Mastery tracking system
- Weak words filtering
- Local storage persistence
- Responsive UI

## Built With

- React
- Vite
- JavaScript
- CSS
- Dictionary API
- Translation API
